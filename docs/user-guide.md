### Daily Tasks Management

#### Creating a New Task
1. Click "Add Task" button
2. Fill in:
   - Task name
   - Description (optional)
   - Date
   - Status (New/In Progress/Completed)
3. Click "Save" to create the task

#### Adding Subtasks
1. Open a task by clicking on it
2. Click "Add Subtask"
3. Fill in subtask details
4. Optional: Add a checklist to track progress
   - Click "Add Checklist"
   - Add items to your checklist
   - Check items as you complete them

#### Using Checklists
- Add checklist items to break down your task
- Check items as you complete them
- Status updates automatically:
  - No items checked â†’ NEW
  - Some items checked â†’ IN PROGRESS
  - All items checked â†’ COMPLETED

### Kanban Board View

#### Using the Kanban Board
1. Click "Kanban" in the side menu
2. View tasks organized by status:
   - New tasks
   - In Progress tasks
   - Completed tasks
3. Drag and drop tasks between columns to update their status

#### Task Management in Kanban
- Click a task to view details
- Edit task details directly from the card
- Use the checklist to track progress
- Search tasks using the search bar

### Calendar View

#### Navigating the Calendar
1. Click "Calendar" in the side menu
2. Use arrows to move between weeks
3. Click "Today" to return to current week
4. Select any date to see its week number

#### Planning with Calendar
- View tasks scheduled for specific dates
- Click on dates to see details
- Week numbers help with planning
- Search for specific dates
